## 🎯 О проекте

Это игровая платформа где пользователи в реальном времени делают микропредсказания на спортивные события, получают очки и соревнуются в таблице лидеров. Полностью бессерверная архитектура, поддерживает сотни одновременных игроков в одной комнате с задержкой < 100мс.

### ✅ Ключевые принципы:
1. **Single Durable Object** - вся логика игры находится в одном экземпляре
2. **Гарантированная консистентность** - нет гонок данных
3. **Отказоустойчивость** - если воркер упадёт, состояние полностью восстановится
4. **Глобальная доступность** - работает в 300+ точках присутствия Cloudflare
5. **Нулевая стоимость** - полностью помещается в бесплатный тариф

---

## 🎮 Механика игры

1. Каждые **3 секунды** появляется новое событие
2. У игроков **8 секунд** чтобы сделать выбор А или Б
3. По истечению времени событие автоматически разрешается
4. Очки начисляются по формуле:
   ```
   Очки = 100 * (1 - консенсус) * (1 + 0.1 * серия)
   ```
5. Чем меньше людей выбрало твой вариант - тем больше очков ты получаешь
6. За каждую подряд правильную догадку даётся +10% бонус

## 🚀 Запуск локально

```bash
# Клонируем репозиторий
git clone https://github.com/a197428/Realtime-Micro-Predictions-Cloudflare-MVP-
cd Realtime-Micro-Predictions-Cloudflare-MVP-

# Устанавливаем зависимости
npm install

# Запускаем dev сервер
npm run dev
```

Приложение будет доступно по адресу `http://localhost:8787`

## ☁️ Деплой на Cloudflare

```bash
# Логин

# Деплой (из корня репозитория)
npm run deploy
```

После деплоя приложение будет доступно по всему миру.
